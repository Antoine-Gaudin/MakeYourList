/tarifs — L'utilisateur voit les plans

Nouveau parcours — 3 étapes
1
/tarifs — L'utilisateur voit les plans et clique "S'abonner"
2
Modale s'ouvre sur la même page — inscription ou connexion rapide
3
Soumission → direct vers Stripe pour le paiement
Résultat : L'utilisateur ne quitte jamais la page tarifs. Moins de friction = plus de conversions.

Comment ça marche concrètement ?

Quand un visiteur non connecté clique "S'abonner", une modale s'ouvre directement sur la page tarifs.
Il peut s'inscrire ou se connecter sans quitter la page.
Une fois authentifié, il est immédiatement redirigé vers Stripe pour finaliser le paiement.

Note : La confirmation d'email reste obligatoire pour des raisons de sécurité, mais elle peut se faire après le paiement — l'utilisateur n'est pas bloqué.
